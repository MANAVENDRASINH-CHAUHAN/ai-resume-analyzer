# 📄 AI Resume Analyzer & Matcher 🚀

## 📌 Overview
AI Resume Analyzer is a smart web application that analyzes resumes and matches them with job requirements.  
It helps to companies find skilled candidates.
---

## ✨ Features

### 👤 User Features
- Upload resume (PDF/DOC)
- Analyze resume content
- Get skill extraction
- View match score with job role

### 🤖 AI Features
- Resume parsing using AI
- Skill extraction from resume
- Job-role based matching
- Suggest improvements for resume

### 🧑‍💼 Admin / Recruiter Features
- View uploaded resumes
- Compare candidates
- Analyze multiple resumes
- Shortlist based on score
